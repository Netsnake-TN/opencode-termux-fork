# opencode-termux
OML/OpenCode on Termux (OCT)
紧急发布第一个暂未发现与linux版本行为异常的构建包，源代码将在几天后（预估）上传。当前仍在GPL和MIT协议徘徊中，请提供建议，谢谢大家。

The emergency release of the first build package that has not yet found an abnormal behavior with the Linux version, the source code will be uploaded in a few days (estimated).The GPL and MIT agreement are still hovering, please provide advice, thank you.

这是一个 oh-my-litecode 的子项目，构建了bun.js和opencode的安卓Termux运行时。 目标是追踪最新版本，目前与 oh-my-opencode 通过一些微调兼容。bun-termux随项目构建，作为依赖。是否需要将其分离？ 请提供建议，谢谢！

This is a oh-my-litecode subproject that builds the Android Termux runtime of bun.js and opencode. The goal is to track down the latest version, which is currently compatible with oh-my-opencode with some fine-tuning. bun-termux is built with the project, as a dependency.Does it need to be separated? Please provide advice, thank you!
